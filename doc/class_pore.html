Detailed Description

represents one pore

DESCRIPTION

Class Pore represents the single pore in a porous material. Is has attributes such as diameter and length (the pore is represented here as a tube).

◆ calculate_maximal_length()

void Pore::calculate_maximal_length ( Network *  S = NULL,
double  l_max = 10,
double  l_0 = 1 
)

calculate maximal pore length

This function returns the maximal length the pore can obtain taking into account geometric constrains.

WARTNING: Periodic boundary conditions are traced correctly for "hexagonal" and "triangulation" type of topology. Other geometries need their own tracing of periodicity.

ATTENTION: Maximal length can be exceed by the pore when merging in on – the geometric constrains are less important to us than the local reaction area and resistivity of the system

Parameters
Spointer to the network
l_maxmaximal length for the network, important for periodic boundary conditions
l_0characteristic pore's length, if there is a problem with calculation max length l_0 is returned
